# Usage
```
└─$ python3 decode_uac.py      
Usage: decode_uac.py <UAC_VALUE>
  Example: decode_uac.py 66048
  Example: decode_uac.py 0x10200
```
```
└─$ python3 decode_uac.py 66082
UserAccountControl: 66082 (0x00010222)
  [+] ACCOUNTDISABLE
  [+] PASSWD_NOTREQD
  [+] NORMAL_ACCOUNT
  [+] DONT_EXPIRE_PASSWORD
```
